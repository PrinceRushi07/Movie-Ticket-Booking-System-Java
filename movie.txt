This is a simple yet effective **Movie Ticket Booking System** implemented entirely using Java and the ArrayList collection. The application mimics basic real-world booking system functionality such as:

1. Adding multiple movies
2. Booking a movie ticket (user selects a movie)
3. Viewing all bookings
4. Checking if a user has already booked
5. Displaying a sublist (range) of bookings
6. Cancelling a booking
7. Displaying the first and last booking
